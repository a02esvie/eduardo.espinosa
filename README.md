# Web académica de Eduardo Espinosa Víctor

Este prototipo es una web estática lista para revisar en navegador y preparada para publicarse en GitHub Pages.

## Archivos

- `index.html`: página principal.
- `styles.css`: estilos visuales.
- `assets/research-hero.png`: imagen generada como recurso visual de apoyo.

## Publicación gratuita recomendada

1. Crear un repositorio público en GitHub.
2. Subir el contenido de esta carpeta.
3. Activar GitHub Pages desde la rama principal.
4. Revisar la web publicada en la URL `https://usuario.github.io/repositorio/`.

También se puede trasladar el contenido a WordPress.com, Google Sites o Notion usando el archivo `contenido-web-académica-eduardo.md`.
